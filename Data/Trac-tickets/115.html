Description

I created an updated version of the CF-SWEET mapping spreadsheet, which may be useful to others. I'd like to add it to the CF site eventually, when an appropriate location becomes available. Meanwhile, thie ticket makes it available to people while we sort out what to do about it.

The primary purpose of the document was to break down CF standard names into their constituent components, based on the CF-SWEET mapping done by Raskin et al around 2010. Additional features:

  • updated for v26 of standard names
  • generation of an autocalculated name, based on ordering the component values so that most names came out OK
  • counted total number and number unique terms in each component type
  • referenced the # of SWEET terms in some categories (from the SWEET ontology), for comparison
  • tracked the unique terms in each category (at the bottom of the table)
  • highlighted some changes and issues

Highlight key includes:

  • yellow cells indicate new parameters
  • light blue cells indicate additions to (existing?) entries, possibly correcting previous mappings
  • light and dark orange cells indication corrections
  • red triangle in upper right corner indicates an Excel comment about some issue
