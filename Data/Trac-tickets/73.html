#73 accepted defect

Rename Appendix G as Appendix Z

Reported by: jonathan Owned by: davidhassell
Priority: medium Milestone:
Component: cf-conventions Version:
Keywords: Cc:

Description

Appendix G is the revision history of the CF standard document. When first added it was at the end of the document, which is a convenient place for it, but in CF 1.6 there is an Appendix H following it. For future convenience, I propose that Appendix G should be renamed Appendix Z in the next release, so that it returns to the end of the document and stays there forever.

Jonathan
