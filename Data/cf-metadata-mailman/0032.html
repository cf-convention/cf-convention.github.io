A few improvements to the CDM Validator are at:

  http://motherlode.ucar.edu:9080/thredds/cdmValidate.html

including ability to check files remotely. These can be files on an
OPeNDAP server or on an HTTP 1.1 server that supports Byte Ranges.

You can also get the output back as XML (format still evolving some).

Comments are welcome.
