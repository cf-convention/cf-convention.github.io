Hi all,

CMOR (used to rewrite nearly all "IPCC" CMIP3 output in compliance with
CF) has an option allowing the user to use udunits to convert from the
user's "native" units to the standard units mandated for CMIP3.

I know this is not quite like software needing to read and transform
units, but if we allow non-compliant udunits, CMOR will not be able to
help the user write data in this way.

cheers,
Karl
