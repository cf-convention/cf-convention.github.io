Howdy all!

Libcf-1.0 alpha4 has just been released, and can be found here:
http://www.unidata.ucar.edu/software/libcf/

The alpha4 release offers the C/Fortran programmer some assistance
with reading or writing CF Convention files. The C API is tested
reasonably well - the fortran wrappers are present but hardly tested.

An alpha5 release is planned before the end of the year. It will
include the capability to read a geographically defined subset of data
from a CF file.

Documentation for the C API can be found here:
http://www.unidata.ucar.edu/software/libcf/docs/libcf

Comments are welcome, please send them to: ed at unidata.ucar.edu.

The API of libcf is still under active development. Suggestions for
useful features are very welcome.

Thanks!

Ed
