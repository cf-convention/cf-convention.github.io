Hi,

The EU funded project Damocles (http://damocles.met.no/data_management/) has
been using NetCDF/CF as the file format standard for both gridded (CF) and
other datasets (~CF). An metadatabase with tools that automatically checks
and extracts metadata from NetCDF/CF files have been developed and is
available at https://wiki.met.no/metamod/. This software is being used for
IPY data management activities as well.
