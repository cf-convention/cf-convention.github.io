I overlooked a problem here. We already have a name of "heat diffusivity"
in the atmos so we should use that, for consistency, instead of "temperature
diffusivity". Therefore probably it should be "salt" instead of "salinity" too.

If NEMO would like to be most helpful, it should probably call the tracer
diffusivities "tracer" when they are identical, and "heat" and "salt" when
they aren't. Jonathan
