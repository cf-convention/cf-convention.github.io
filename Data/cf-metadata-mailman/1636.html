Dear All,

The standard name table has been updated with the changes previously
announced. The current version is now version 10, dated 21 October
2008.

The next scheduled update for the standard name table will take place on
Tuesday 11 November. The changes to the standard name table resulting
from the acceptance of trac ticket 17 will be made on that day. This
means that all standard names using the "where" phrase will be made
aliases of the equivalent names without "where" and in future the
information regarding surface type should be recorded in cell_methods.

Best wishes,
Alison
