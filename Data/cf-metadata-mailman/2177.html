The original UDUNITS package doesn't support logarithmic units. The
replacement UDNITS-2 package *does* support logarithmic units --
including "dBZ".

The UDUNITS-2 package doesn't have a Fortran interface, however. If you
are a Fortran-2003 expert and would like to collaborate to create an
interface module, then I would love to hear from you.
