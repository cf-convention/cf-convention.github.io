Dear Jonathan,

Thank you for pointing out the typo. The name with the incorrect
spelling appears as far back as version 1 of the standard name table,
which means that it was probably like that even before we migrated to
the present CF website. In view of the length of time this name has
been around (at least 3 years), I will make the correction by creating
an alias in the next table update so as not to invalidate any data that
may have been written with the misspelled version.

Best wishes,
