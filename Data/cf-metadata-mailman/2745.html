Dear All,

There is a need to introduce a number of new standard names to describe
the output from sea ice models which will form part of the CMIP5
project. The attached spreadsheet lists the proposed name, unit and a
brief explanation for each new model output quantity.

The list of quantities and their explanations are taken from the most
recently published version of the CMIP5 outputs document, dated 8 July
2010 - see
http://cmip-pcmdi.llnl.gov/cmip5/output_req.html?submenuheader=2#req_lis
t. Siobhan O'Farrell coordinated the work to define the list of required
sea ice outputs and she proposed some new standard names in 2009. Since
Siobhan's original posting the list seems to have evolved, hence this
revised set of proposals.

I have tried as far as possible to ensure consistency between the
current proposals and existing standard names. I found the summary of
sea ice formation processes at
http://nsidc.org/seaice/characteristics/formation.html helpful in
understanding some of the quantities and I will use this information in
writing the definitions for the standard name table. I am unsure of the
meaning of the quantities in rows 29-31 and would welcome any help in
clarifying the explanations.

Comments on all these names are now invited.

Best wishes,
Alison
