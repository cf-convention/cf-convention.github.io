The measurement in question were HF-Radar measurements.
With a combination of different stations a surface velocity vector is
measured.
The measurements are stored as a grid in a netcdf file with the variables
(standard names):
surface_eastward_sea_water_velocity
surface_northward_sea_water_velocity

To visualize the measured data ncWMS/Godiva2 by Jon Blower is used.
This tools takes the components and creates a new display variable
surface_sea_water_velocity which is displayed as a vector.
The automatically created vector surface_sea_water_velocity doesn't exist
in the standard name table.

It would be nice if this vector could be added to the standard name table.
