Dear All,

The standard name table has been updated today as planned. The current version is now version 17, dated 24 March 2011. While every effort has been made to ensure that the names, units and definitions are correct, please let me know if you do spot any mistakes or omissions.

N.B. These names should now have been moved to the vocabulary editor 'completed work' status page, but for some reason they are still being listed as 'current work'. Steve Donegan is currently working to address this and the editor pages should be updated shortly.

Best wishes,
Alison
