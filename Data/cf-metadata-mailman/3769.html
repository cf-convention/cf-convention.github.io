Hi Ian,

The CF Checker using UDUNITS-2 to validate the units.

This is another one of those units where UDUNITS-1 deemed it valid and
UDUNITS-2 says it's invalid.

I think the CF Document should be updated to specify validation against
UDUNITS-2 and also to remove the reference to the udunits.dat file as
this no longer exists at UDUNITS-2, instead xml files are used.

Regards,
Ros.
