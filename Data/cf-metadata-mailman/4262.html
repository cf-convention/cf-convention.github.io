Hi Christian,

These names look fine (I think there's a missing underscore in the
first) so we would have:

upward_sensible_heat_flux_in_air; W m-2
upward_latent_heat_flux_in_air; W m-2.

The syntax is consistent with many existing names and we already have
surface fluxes for the same quantities. I assume that these are 3-D
quantities and you would use a vertical coordinate variable to specify
the vertical location of the flux.

These names are accepted for inclusion in the standard name table. I
have already committed the changes for today's update, however, another
update will take place one month from now and it will include these
names.

Best wishes,
Alison
