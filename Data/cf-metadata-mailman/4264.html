Dear All,

The Standard Name Table has been updated on the CF website to Version 19, dated 22 March 2012. In addition, the following names have been added to the list of contributors to CF Standard Names (http://cf-pcmdi.llnl.gov/documents/cf-standard-names/cf-standard-name-contributors): Martin Schultz; Rainer Feistel; Trevor McDougall; Jim Biard; Kevin Tansey. (Paul Durack is already among our list of contributors.)

Unfortunately there has been a problem in sending the changes to the NERC vocabulary server so they are not yet published there. When the problem has been resolved I will change the date stamp of the copy on the CF website to avoid any confusion/conflict between the two. Sorry for any inconvenience but I did not want to delay the whole update because of a technical issue.

There will be another update of the Standard Name Table one month from now.

Best wishes,
Alison

> The standard name table will be updated tomorrow with new names,
> updated definitions and units that have been agreed over recent months.
> Details of the planned changes are given at the end of this message.
> The changes will be submitted to the NERC vocabulary server today and
> should be published overnight. Tomorrow I will update the xml and html
> versions of the standard name table on the CF website. In this way the
> date and version information in the two locations will be kept
> synchronised. I will post again when the update is complete.
>
> A number of proposals are still under discussion and these will NOT be
> included in the update tomorrow. Many of them are quite close to
> agreement, but I have to draw the line somewhere in order to finalise
> the list of names and definitions that will go into the table this time
> around. I will make another update to the standard name table in one
> month's time and names that are agreed in the intervening period will
> be added at that stage. The names that are not yet finalised are:
>
> 1)
> volume_extinction_coefficient_in_air_due_to_mie_scattering_of_ambient_a
> erosol; m-1 (proposed by Martin Schultz).
> There is still some discussion centred on
> extinction/scattering/attenuation terminology. See
> http://mailman.cgd.ucar.edu/pipermail/cf-metadata/2012/027812.html for
> latest posting.
>
> 2) Emission names (proposed by Martin Schultz).
> We have now agreed on the syntax of these names and the new chemical
> species to be added. There has been some discussion of the names and
> definitions of emissions sectors. I want to allow a few more days to
> finalise this discussion, but I think we are nearing agreement on the
> sectors originally proposed by Martin. See
> http://mailman.cgd.ucar.edu/pipermail/cf-metadata/2012/027814.html for
> latest posting.
>
> 3) New surface aerosol optical properties names (proposed by Markus
> Fiebig).
> See trac ticket 81 for proposal and
> http://mailman.cgd.ucar.edu/pipermail/cf-metadata/2012/027835.html for
> latest posting.
>
> Best wishes,
> Alison
