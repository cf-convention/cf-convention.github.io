Hi folks,

We've had various discussions about how one might be able to use
netcdf-JAVA from Python.

Here is new option:

http://pyjnius.readthedocs.org/en/latest/index.html

Maybe one of the folks that cross the Java/Python worlds could give it a try?

(I do note from a blog discussion that it does not currently have an
efficient way to move large arrays of data back and forth, which would
be critical to this use, but if it works at all, that may not be all
that hard to add -- it's built with Cython, which has great numpy
support)
