Dear All,

The standard name table has been updated today on the CF website (http://cf-pcmdi.llnl.gov/documents/cf-standard-names). The current version is now version 20 dated 11 September 2012. A total of 240 new names have been added - a full list is given at the end of this message. Christian Wetzel has been added to the list of contributors to CF Standard Names (http://cf-pcmdi.llnl.gov/documents/cf-standard-names/cf-standard-name-contributors) (Martin Schultz and Stephane Tarot are already listed as contributors).

Unfortunately there has been a problem in sending the changes to the NERC vocabulary server so they are not yet published there. When the problem has been resolved I will change the date stamp of the copy on the CF website to avoid any confusion/conflict between the two. Sorry for any inconvenience but I did not want to delay this update any longer due to technical issues as it was originally scheduled for 30 August.

I plan to make another update to the standard name table in approximately one month's time.

Best wishes,
Alison
