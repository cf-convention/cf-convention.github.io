All,

This is an extension of recent discussions about the mixed
Julian/Gregorian calendar. Here is a new ticket proposing an explicit
CF calendar name "Julian/Gregorian". This is for data sets that have
a legitimate need to span the Julian/Gregorian discontinuity with the
Udunits-style calendar, and also to facilitate backward compatibility
with existing data sets.

   https://cf-pcmdi.llnl.gov/trac/ticket/96

Comments welcome. Please post related discussion on the ticket, not
on the general mailing list. Pursuant to CF rules, I request to be
appointed as the ticket moderator.

--Dave
