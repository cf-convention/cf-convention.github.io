The standard name table will be updated later today with the list of changes detailed below. I will post again when the update is complete. Please note that there are also a number of open mailing list threads in which standard names are currently under discussion. Those proposals are not included in today's update.

Best wishes,
Alison

2. Clarification to definitions of existing names

a. toa_bidirectional_reflectance.

The definition will be updated following discussion of a question asked by Randy Horne. The following text will be added to clarify the meaning:
 ' toa_bidirectional_reflectance includes a factor to account for the cosine of the solar zenith angle but does not include any integration over solid angle.'

b. tendency_of_atmosphere_mass_content_of_X_due_to_emission_from_industrial_processes_and_combustion names. Proposed by martin Schultz and Angelika Heil.

22 names of this pattern were recently added to the standard name table (version 20, 12 September 2012). The definitions contained a typo in the list of IPCC emissions categories: category 2E, "Production of halocarbons SF6", is being replaced by category 2G " Non-energy use of lubricants/waxes (CO2)".
