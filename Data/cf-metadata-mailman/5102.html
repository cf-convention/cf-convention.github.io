we are measuring global radiation and photosynthetic active radiation
using the standard names surface_downwelling_shortwave_flux_in_air and
surface_downwelling_photosynthetic_photon_spherical_irradiance_in_air.

find a standard name for this kind of radiation. Is there a way of
construction a standard name from existing standard names (something like
downwelling_spectral_radiative_flux_in_air_integrated_from_100_to_400nm)
or should a new standard name get defined?
