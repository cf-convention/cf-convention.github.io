Dear All,

The standard name table was updated on Friday with the list of changes detailed below. The current version of the table is now version 24, dated 27 March 2013. The changes have also been published on the NERC vocabulary server.

In addition to publishing many of Aleksandar Jelenak's recent proposals for satellite standard names, this update includes a number of names that were agreed some time ago but had not been added to the table. Names arising from the discussion under trac tickets 93 and 37 have been added.

It is an unusual move, but I intend to make another 'mini' update to the standard name table next week. This is because of the two names that were agreed in CF trac ticket #37 (the discrete sampling geometries ticket), see 1c below. The names that were eventually agreed were different from the 'working versions' that had been under discussion for some time. As a result, Jonathan requested that the names and their aliases both go into the table at the same time, in case test data had been written using the original versions. The usual method by which I add aliases to the NERC vocabulary server makes it difficult to create the two versions of the names simultaneously. Since only two names are affected, and since this is unlikely to be a regular occurrence, the simplest solution is to add the original versions of the names in this update and follow up in a day or two with a second update to create the aliases.

Aleksandar Jelenak, Bruno Piguet and Gunther Lang have been added to the list of standard name contributors (http://cf-pcmdi.llnl.gov/documents/cf-standard-names/cf-standard-name-contributors). The other contributors to this update were already listed.

Best wishes,
Alison
