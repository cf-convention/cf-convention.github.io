Dear All,

As notified last week, a 'mini' update has been applied to the standard name table, mainly to create the aliases agreed in trac ticket 37. The current version of the table is now version 25, dated 5 July 2013. The changes have also been published on the NERC vocabulary server. The full list of changes is detailed below.

Best wishes,
Alison

1. New names

No new names were added in this update.

2. Aliases for existing names.

Aliases for names agreed under trac ticket #37 (discrete sampling geometries).
platform_id (string valued therefore unitless)
platform_name (string valued therefore unitless)

3. Modifications to definitions of existing names

The definitions of 13 existing platform names have been modified to add 'satellite' to the list of platform types. In addition, the definition of platform_zenith_angle was modified to cross-reference the recently added sensor_zenith_angle name.

platform_zenith_angle (degree)
platform_yaw_rate (degree s-1)
platform_yaw_angle (degree)
platform_speed_wrt_sea_water (m s-1)
platform_speed_wrt_ground (m s-1)
platform_speed_wrt_air (m s-1)
platform_roll_rate (degree s-1)
platform_roll_angle (degree)
platform_pitch_rate (degree s-1)
platform_pitch_angle (degree)
platform_orientation (degree)
platform_course (degree)
angle_of_rotation_from_solar_azimuth_to_platform_azimuth (degree)
