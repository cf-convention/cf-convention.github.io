Dear Petra,

I have created such a checker for similar purposes. We are using it on
the command line as well as on the web. It supports CF 1.0 to 1.5.
Support for 1.6 is in development.
The checker is written in purely in python with a few relatively easily
installable dependencies (see README).
Have a look at https://bitbucket.org/mde_/cfchecker for more details.

Regards,
Michael
