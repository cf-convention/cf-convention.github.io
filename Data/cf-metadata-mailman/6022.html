Hello,

I have a NetCDF file with the time coordinate variable's calendar set specifically as 'Gregorian'. The cfchecker tool has no complaint about this but another piece of software does not allow this calendar because the string is not all lowercase. Checking the documentation (http://cf-pcmdi.llnl.gov/documents/cf-conventions/1.7-draft1/cf-conventions.html#calendar) implicitly states that calendar strings should be entirely in lowercase but does not confirm this.

As such might I request some clarification as to whether 'Gregorian' is also a valid calendar string, and by extension whether such capitalisation is likewise acceptable in other calendar strings?

Many thanks,
