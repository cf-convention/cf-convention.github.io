Before responding to Jim, can I propose this for the second sentence: "Missing data are permitted in an auxiliary coordinate variable only at points where all the referencing variables also have missing data." Trying to fix some nits and make it a bit more parseable.

Jim, I'm not sure I understand your question. If you have data points for a variable ('D') where you do not have data for the auxiliary coordinate variable ('A'), than you have to either (a) somehow derive the corresponding auxiliary coordinate variable (A) data before creating the file, or (b) not specify the intended auxiliary coordinate variable A as an auxiliary coordinate variable for that data variable D. The intent being that it's only an auxiliary coordinate variable if you have a datum for every referencing data point.

On the other hand, if you have _extra_ data in the auxiliary coordinate variable A, but no corresponding data in the regular data variable D, all is fine, it doesn't matter, no action is needed.
