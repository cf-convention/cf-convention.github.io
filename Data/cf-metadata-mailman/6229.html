Dear All:

I would like to propose the following standard_name:

dvorak_tropical_cyclone_cloud_region_scene_type

definition:
The variable is a string, taking on one of the following values to indicate the Advanced Dvorak Technique tropical cyclone cloud region scene type: uniform_central_dense_overcast, embedded_center, irregular_central_dense_overcast, curved_band, or shear. For a data variable, it is encoded as an integer using flag_values and flag_meanings.
 
canonical units:
1


very respectfully,

randy
