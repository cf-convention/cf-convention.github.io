Dear All:
  
 I would like to propose the following standard_name:
  
 dvorak_tropical_cyclone_eye_region_scene_type
  
 definition:
  
 The variable is a string, taking on of the following values to indicate
the Advanced Dvorak Technique tropical cyclone eye region scene type:
clear_ragged_or_obscured_eye, pinhole_eye, large_eye, or no_eye. For a
data variable, it is encoded as an integer using flag_values and
flag_meanings.
  
 canonical units: 1
  
  
 very respectfully,
  
 randy
