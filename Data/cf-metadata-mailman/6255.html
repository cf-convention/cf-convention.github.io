Dear All:
  
 In concert with an amendment made to a sister tropical cyclone related
standard name based on feedback received
(scene_type_of_dvorak_tropical_cyclone_eye_region), I would like to amend
this standard_name in an analogous manner (i.e. start the name with
"scene_type_of")
  
 Summarizing the amendment:
  
 standard_name:
 scene_type_of_dvorak_tropical_cyclone_cloud_region
  

 definition: The variable is a string, taking on one of the following
values to indicate the Advanced Dvorak Technique tropical cyclone cloud
region scene type: uniform_central_dense_overcast, embedded_center,
irregular_central_dense_overcast, curved_band, or shear. For a data
variable, it is encoded as an integer using flag_values and flag_meanings.
 canonical units: 1
 very respectfully,
  
 randy

  

----------------------------------------

 Dear All: I would like to propose the following standard_name:
dvorak_tropical_cyclone_cloud_region_scene_type definition: The variable
is a string, taking on one of the following values to indicate the Advanced
Dvorak Technique tropical cyclone cloud region scene type:
uniform_central_dense_overcast, embedded_center,
irregular_central_dense_overcast, curved_band, or shear. For a data
variable, it is encoded as an integer using flag_values and flag_meanings.
 canonical units: 1 very respectfully, randy
