Good question Reyna! To my knowledge there is no defined relationship between the CF Standard Name table versions, and the CF Convention versions.

You could derive the time relationships by determining the publication dates of each. There would be likely to be some imprecision in that determination. But no one says that a CF file following a particular convention has to use the current standard names table, as opposed to an old one, or one from the future.

John
