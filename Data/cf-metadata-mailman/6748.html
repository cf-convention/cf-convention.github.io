We have had a continuing spam problem on the Trac issue tracker for the
CF Conventions, http://cf-trac.llnl.gov/trac.
Today I deleted over 1600 users whom I believe to be illegitimate, and
closed 14 spam "tickets".

If you have used the CF Trac system in the past, you should see no
change in how you use it.

But for new people: If you register as a new user, you will have to ask
me to put you in the system, before you can add or modify any ticket.
In the near future I'll add a note about this to the registration page.

I'm sorry for the inconvenience.
