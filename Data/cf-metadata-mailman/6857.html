In OceanSITES, we use a standard name 'whenever one is available' for a
given variable.

You can use the 'coordinates' attribute to tell users which latitude to
use; it
points to the correct variable by short name, so it lets you advise the
user on
which is the most appropriate of the variables that share a standard name.

As far as I know, most software recognizes this. If not, we have a problem,
or the developers do.

Cheers - Nan
