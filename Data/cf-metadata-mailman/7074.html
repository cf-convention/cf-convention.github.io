Dear all

The bulk of archived CF-netCDF salinity data is probably the vast amounts of
CMIP ocean model data, so I don't think we can reinterpret what this means
more precisely, since models differ in which equations of state they use. Thus
sea_surface_salinity and sea_water_salinity are deliberately vague concepts,
because that suits the purpose (GCMs) for which those names were originally
introduced. However, it's also fine to have more precise quantities, obviously.

Best wishes

Jonathan
