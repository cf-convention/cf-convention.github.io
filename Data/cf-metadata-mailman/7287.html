Dear all,

We have moved our online CF-Checker to a new service at CEDA. It now runs as a process inside our Web Processing Service.

You can access the web form to submit files to the checker here:

 http://wps-web1.ceda.ac.uk/submit/form?proc_id=CFChecker

It uses the "cf-checker" code managed in GitHub at:

 https://github.com/cedadev/cf-checker/

Please let me know if you have any problems with the online checker or if you have any other feedback.

Kind regards,
