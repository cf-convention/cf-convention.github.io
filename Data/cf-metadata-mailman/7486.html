Dear All,

The standard name table has been updated with the changes listed at the end of this message.

The current version of the table is now 31, dated 8 March 2016. The links on the CF website have been updated accordingly and the standard name changes have also been published on the NERC vocabulary server.

Links to the list of people who have contributed to the standard names over the years have been added to the following pages: http://cfconventions.org/documents.html and http://cfconventions.org/standard-names.html. Many thanks to Jeff Painter and Matthew Harris at PCMDI for helping to track down the original file from the old CF website. The list should now be up to date, but apologies if I have accidentally left out anyone's name. Please let me know if you spot any errors or omissions.

The next update to standard names will take place on 5 April.

Best wishes,
Alison

1. New names

acoustic_signal_roundtrip_travel_time_in_sea_water (canonical units: s)
change_in_land_ice_amount (canonical units: kg m-2)
downward_air_velocity (canonical units: m s-1)
dry_atmosphere_mole_fraction_of_carbon_dioxide (canonical units: 1)
dry_atmosphere_mole_fraction_of_methane (canonical units: 1)
eastward_land_ice_velocity (canonical units: m s-1)
northward_land_ice_velocity (canonical units: m s-1)
number_of_days_with_surface_temperature_below_threshold (canonical units: 1)
single_scattering_albedo_in_air_due_to_ambient_aerosol_particles (canonical units: 1)

2. Modification to definitions of existing names

The definition of upward_air_velocity has been modified to cross reference the new name downward_air_velocity.

3. Modification to units of existing names

A correction has been made to the units of one name after Martin Juckes reported an error:
tendency_of_specific_humidity_due_to_stratiform_cloud_and_precipitation_and_boundary_layer_mixing (Canonical units change from K s-1 to s-1)

4. New aliases for existing standard names

No new aliases are included in this update.
