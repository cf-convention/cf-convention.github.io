Dear All,

The standard name table has been updated with the changes listed at the end of this message. (This was a relatively small update).

The current version of the table is now 34, dated 13 June 2016. The links on the CF website and list of standard name contributors have been updated accordingly; the standard name changes have also been published on the NERC vocabulary server.

The next update to standard names will take place on 19th July.

Best wishes,
Alison

1. New names

Proposed by Tuomas Karna:
specific_turbulent_kinetic_energy_dissipation_in_sea_water (W kg-1)
specific_turbulent_kinetic_energy_of_sea_water (m2 s-2)
turbulent_mixing_length_of_sea_water (m)

2. Modifications to definitions of existing names

Proposed by Lars Barring:
The definitions of both the following names were clarified to emphasise that only positive values of temperature deficit/excess are included in the integrals. Also, a sentence was added to state that the threshold temperature against which the deficit/excess is calculated should be provided in a scalar or size one coordinate variable with standard name air_temperature_threshold.
integral_of_air_temperature_deficit_wrt_time
integral_of_air_temperature_excess_wrt_time
