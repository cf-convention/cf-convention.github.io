Dear All,

I have realized that I neglected to announce the most recent update of the standard name table, so here it is, if a little late.

The standard name table has been updated with the changes listed at the end of this message.

The current version of the table is now 35, dated 20th July 2016. The links on the CF website and the list of standard name contributors have been updated accordingly; the standard name changes have also been published on the NERC vocabulary server.

The next update to standard names will take place on 13th September. There will be no update in August to allow for annual leave.

Best wishes,
Alison

1. New names

Proposed by Elodie Fernandez and Roy Lowry in the mailing list thread "Waves":
sea_surface_primary_swell_wave_significant_height (m)
sea_surface_secondary_swell_wave_significant_height (m)
sea_surface_wave_maximum_crest_height (m)
sea_surface_wave_maximum_height (m)
sea_surface_wave_maximum_trough_depth (m)
sea_surface_wave_mean_height (m)
sea_surface_wave_mean_height_of_highest_tenth (m)

2. Modifications to definitions of existing names

Proposed by Elodie Fernandez and Roy Lowry in the mailing list thread "Waves":
The definition of wave significant height was clarified for three existing names by adding the sentence "Significant wave height is a statistic computed from wave measurements and corresponds to the average height of the highest one third of the waves, where the height is defined as the vertical distance from a wave trough to the following wave crest."
sea_surface_swell_wave_significant_height (m)
sea_surface_wave_significant_height (m)
sea_surface_wind_wave_significant_height (m)

No new aliases were added in this update and there were no changes to units of existing names.
