As part of the RFMIP, I would like to propose the standard names for new
variables listed below in the following format: Short Name, Proposed
CFStandard Name, Description, and Units. I welcome feedback from the CF
list on this matter.

Thanks so much,
Dan

Short Name: rsutcc
CFStandard Name: toa_outgoing_shortwave_flux_assuming_clean_clear_sky
Description: Calculated in the absence of aerosols and clouds. This
requires a double-call in the radiation code with precisely the same
meteorology.
Units: W/m^2

Short Name: rsdscc
CFStandard
Name: surface_downwelling_shortwave_flux_in_air_assuming_clean_clear_sky
Description: Calculated in the absence of aerosols and clouds. This
requires a double-call in the radiation code with precisely the same
meteorology.
Units: W/m^2

Short Name: rsuscc
CFStandard
Name: surface_upwelling_shortwave_flux_in_air_assuming_clean_clear_sky
Description: Calculated in the absence of aerosols and clouds. This
requires a double-call in the radiation code with precisely the same
meteorology.
Units: W/m^2

Short Name: rsucc
CFStandard Name: upwelling_shortwave_flux_in_air_assuming_clean_clear_sky
Description: Calculated in the absence of aerosols and clouds. This
requires a double-call in the radiation code with precisely the same
meteorology.
Units: W/m^2

Short Name: rsdcc
CFStandard Name: downwelling_shortwave_flux_in_air_assuming_clean_clear_sky
Description: Calculated in the absence of aerosols and clouds. This
requires a double-call in the radiation code with precisely the same
meteorology.
Units: W/m^2
