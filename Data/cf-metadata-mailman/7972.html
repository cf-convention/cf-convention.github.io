I'd like to propose 4 new standard names as part of the HighResMIP data request for CMIP6.

1. prcsh shallow_conv_precipitation_flux
2. uqint integrated_eastward_wind_times_humidity
3. vqint integrated_northward_wind_times_humidity
4. wbptemp wet_bulb_potential_temperature

1.
Reason:
Many models use both a shallow and a deep convective parameterisation, but until now only the total precipitation from both schemes has been diagnosed. The contribution from deep convection is expected to dominate, and adding this diagnostic will enable more detailed assessment of model processes.

CF standard name
shallow_convective_precipitation_flux

short name
prcsh

Comment:
Analogous to existing name convective_precipitation_flux

Native grid:
atmos

Units
kg m-2 s-1

Long name:
shallow_conv_precipitation_flux

Description:
Convection precipitation from shallow convection

2.
Reason: Column-integrated moisture fluxes will make it much easier to run metrics and diagnostics on many models without the need to do imperfect column integration on different grids.

CF standard name
atmosphere_product_of_eastward_wind_and_specific_humidity

short name
uqint

Comment:
There is an existing standard name of product_of_eastward_wind_and_specific_humidity and we use "atmosphere" in existing names to mean the atmospheric column, so I suggest we combine these into a new standard name. The definition in the CF standard name table will also state that this quantity applies to the entire atmospheric column.

Native grid:
atmos

Units
m2 s-1

Long name:
integrated_eastward_wind_times_humidity

Description:
Column integrated eastward wind times specific humidity

3.
Reason: Column-integrated moisture fluxes will make it much easier to run metrics and diagnostics on many models without the need to do imperfect column integration on different grids.

CF standard name
atmosphere_product_of_northward_wind_and_specific_humidity

short name
vqint

Comment:
There is an existing standard name of product_of_northward_wind_and_specific_humidity and we use "atmosphere" in existing names to mean the atmospheric column, so I suggest we combine these into a new standard name. The definition in the CF standard name table will also state that this quantity applies to the entire atmospheric column.

Native grid:
atmos

Units
m2 s-1

Long name:
integrated_northward_wind_times_humidity

Description:
Column integrated northward wind times specific humidity

4.
Reason: Wet bulb potential temperature is an important diagnostic for extreme weather and climate events, particularly drivers of rainfall events.

CF standard name
wet_bulb_potential_temperature

short name
wbptemp

Comment:

Native grid:
atmos

Units
K

Long name:
wet_bulb_potential_temperature

Description:
Wet bulb potential temperature
