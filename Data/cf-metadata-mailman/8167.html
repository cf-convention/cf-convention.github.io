Dear All,

The standard name table has been updated. The current version is now V42, dated 28 March 2017. The changes have also been published in the NERC Vocabulary Server. The changes are listed in full at the end of this message.

Malcolm Roberts and James Orr have been added to the list of contributors to CF standard names: http://cfconventions.org/Data/cf-standard-names/docs/standard-name-contributors.html.

The next update to standard names is planned for 24th April 2017.

Best wishes,
Alison

2. Aliases
Three new aliases were proposed by Elodie Fernandez and Roy Lowry for the Copernicus Marine Service. The 'zero_uprossing_period' names were replaced by more generic wave period names. This is because wave period statistics can be calculated using a variety of methods so it is more appropriate to use generic standard names and supply the specific method of calculation in another attribute, such as long_name.
sea_surface_wind_wave_zero_upcrossing_period -> sea_surface_wind_wave_mean_period            
sea_surface_wave_zero_upcrossing_period -> sea_surface_wave_mean_period            
sea_surface_swell_wave_zero_upcrossing_period -> sea_surface_swell_wave_mean_period            
3. Modifications to definitions of existing names
Clarifications to the definitions of 6 existing names were proposed by Elodie Fernandez and Roy Lowry for the Copernicus Marine Service. All wave direction name definitions now contain the text 'The direction is a bearing in the usual geographical sense, measured positive clockwise from due north.' In addition, improved descriptions of swell_waves and wind_waves were added for consistency with the recently introduced wave height names. The affected names are:
sea_surface_swell_wave_from_direction             
sea_surface_swell_wave_to_direction             
sea_surface_wave_from_direction                
sea_surface_wave_to_direction             
sea_surface_wind_wave_from_direction               
sea_surface_wind_wave_to_direction
