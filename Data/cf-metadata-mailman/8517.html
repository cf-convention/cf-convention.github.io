I like the "redundant" sentence. I like the _Unsigned idea, but there
are lots of old files "in the wild" that don't have that attribute.

This raises another question in my mind. To what extent does the current
version of the document need to give homage to old usages? If your file
says it uses CF 1.5, I wouldn't expect it to follow CF 1.7. So does the
CF 1.7 or 1.8 document need to contain "grandfather" language (things
that used to be, or not be, but that are different in the current version)?

Grace and peace,

Jim
