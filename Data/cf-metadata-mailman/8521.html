I was unaware of the _Unsigned attribute. I was always confused by the
language around setting the ranges to the expected unsigned range. Until
this discussion I wasn't really sure what that was about. Having used
unsigned types, now I'm beginning to wonder if the data we've created
should be changed. If I understand Charlie correctly, then I agree with
him: I think _Unsigned = true is clearer. I think requiring both is
burdensome and allowing one or the other is crufty. (What a great word.)

Mary Jo
