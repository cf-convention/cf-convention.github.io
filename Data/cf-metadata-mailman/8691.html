On Thu, Feb 15, 2018 at 3:31 AM, <alison.pamment at stfc.ac.uk> wrote:

> Dear All,
>
>
>
> The CF standard name table has been updated to version 49, dated 13th
> February 2018.
>
>
>
> Version 49 includes the addition of 1089 new radioactive isotope names
> proposed by Heiko Klein. Rather than list all these names here, the
> quickest way to view them is to look at the standard name table itself:
> http://cfconventions.org/Data/cf-standard-names/49/build/cf-
> standard-name-table.html and enter 'radioactivity' in the search box.
> These new terms mean we now have just over 4000 current entries in the
> standard name table (4016 to be precise).
>
>
>
> In addition to the new names, the definitions of 16 existing names have
> been updated to replace references to 'bolus advection' with references to
> 'parameterized eddy advection' for consistency with names added in version
> 48 for the CMIP6 OMIP experiment. No new aliases have been created in this
> update.
>
>
>
> All the changes have also been published in the NERC Vocabulary Server:
> http://vocab.nerc.ac.uk/collection/P07/current/
>
>
>
> The next standard name table update will take place in March.
>
>
>

>
