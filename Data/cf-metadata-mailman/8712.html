Dear All,

Today the standard name table has been updated to version 50 and the area type table to version 7. The changes have not yet been published on the NERC Vocabulary Server - I will update you when this has been done. The changes are listed in full at the end of this message.

Chris Jones has been added to our list of standard name contributors: http://cfconventions.org/Data/cf-standard-names/docs/standard-name-contributors.html.

The next update to the standard name table will take place in April.

Best wishes,
Alison 

---
V50 standard name changes
New names - proposed by Chris Jones for the CMIP6 C4MIP experiment
litter_mass_content_of_13C  (canonical units: kg m-2)
litter_mass_content_of_14C  (canonical units: kg m-2)
mass_content_of_13C_in_vegetation_and_litter_and_soil_and_forestry_and_agricultural_products (canonical units: kg m-2)

mass_flux_of_carbon_into_litter_from_vegetation_due_to_mortality (canonical units: kg m-2 s-1)
mass_flux_of_carbon_into_litter_from_vegetation_due_to_senescence (canonical units: kg m-2 s-1)
mass_flux_of_carbon_into_sea_water_from_rivers (canonical units: kg m-2 s-1)
mass_flux_of_carbon_into_soil_from_vegetation_due_to_mortality (canonical units: kg m-2 s-1)

mass_flux_of_carbon_out_of_soil_due_to_leaching_and_runoff (canonical units: kg m-2 s-1)
soil_mass_content_of_13C (canonical units: kg m-2)
soil_mass_content_of_14C (canonical units: kg m-2)
soil_mass_content_of_inorganic_ammonium_expressed_as_nitrogen (canonical units: kg m-2)
soil_mass_content_of_inorganic_nitrate_expressed_as_nitrogen (canonical units: kg m-2)

surface_downward_mass_flux_of_methane_due_to_wetland_biological_consumption (canonical units: kg m-2 s-1)
surface_upward_mass_flux_of_carbon_dioxide_expressed_as_carbon_due_to_emission_from_fires (canonical units: kg m-2 s-1)

water_table_depth (canonical units: m)
No new aliases have been added in V50 of the standard name table. No modifications to existing names, their definitions or units have been made.
---
V7 area type changes
1. New area types - needed for CMIP6 C4MIP, SIMIP and LS3MIP experiments
crops_of_c3_plant_functional_types
crops_of_c4_plant_functional_types
natural_grasses_of_c3_plant_functional_types
natural_grasses_of_c4_plant_functional_types
pastures_of_c3_plant_functional_types
pastures_of_c4_plant_functional_types
sea_ice_ridges
wetland
2. Modifications to definitions of existing area types
I have added the following definition to the natural_grasses area_type, taken from the EU CORINE land surface classification: "CORINE Land Cover - Part 2: Nomenclature" (https://www.eea.europa.eu/publications/COR0-landcover).
' "Natural grasses" means grasses growing in areas of low productivity, often situated on rough or uneven ground. This can include rocky areas, briars and heathland.'
