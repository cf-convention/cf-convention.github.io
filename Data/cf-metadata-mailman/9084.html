Dear All,

The standard name table has been updated. We are currently at version 54, dated 31 May 2018:
http://cfconventions.org/Data/cf-standard-names/current/build/cf-standard-name-table.html.
The standardized region list has also been updated: http://cfconventions.org/Data/cf-standard-names/docs/standardized-region-names.html.
The standard name changes have been published on the NERC Vocabulary Server and the region changes will be added shortly. The full list of changes is given at the end of this message.

The following people have been added to our list of standard name contributors: Daniel Feldman, Ben Kravitz, Elisa Manzini, Robert Pincus (http://cfconventions.org/Data/cf-standard-names/docs/standard-name-contributors.html).

You may have noticed that the standard name table version number has increased from 52 to 54 since the previous changes were announced. This is to cope with publishing some long standing proposals for RFMIP where it seemed advisable to publish the original versions of the names, then immediately make them into aliases of the finalized versions. Please see my recent posting in the RFMIP/GeoMIP thread if you want a detailed explanation.

There are still many names under active discussion - I hope to complete the remaining CMIP6 names in the next 2 - 3 weeks. An increasing list of non-CMIP names is starting to build up - I will address those as soon as possible and thanks to all for your patience in the meantime.

The dates of the next two standard name table updates will be as follows: 11th June and 2nd July.

Best wishes,
Alison

2. New aliases
No new aliases were added in this update.
3. Modifications to definitions of existing names
a. The definition of 'in_sea_water_excluding_solutes_and_solids' was clarified at the suggestion of Martin Juckes: 'The phrase "in_sea_water_excluding_solutes_and_solids" means that the standard name refers only to the chemical compound water and does not include material that may be dissolved or suspended in the aqueous medium.' This change affects 2 existing names.
isotope_ratio_of_17O_to_16O_in_sea_water_excluding_solutes_and_solids
isotope_ratio_of_18O_to_16O_in_sea_water_excluding_solutes_and_solids 
b. Martin Juckes provided a definition for 'river' and this was added to 7 existing names: '"River" refers to water in the fluvial system (stream and floodplain).
mass_flux_of_carbon_into_sea_water_from_rivers 
mass_flux_of_nitrogen_compounds_expressed_as_nitrogen_into_sea_from_rivers 
minus_one_times_water_flux_into_sea_water_from_rivers
salt_flux_into_sea_water_from_rivers 
virtual_salt_flux_into_sea_water_from_rivers
water_flux_into_sea_water_from_rivers 
water_flux_into_sea_water_from_rivers_and_surface_downward_water_flux
c. Nan Galbraith commented (http://mailman.cgd.ucar.edu/pipermail/cf-metadata/2018/060164.html) that it is useful to include explanations of quantities such as ocean_sigma_t and ocean_sigma_theta in the definitions of all standard names that refer to them. The definitions of 7 existing names were updated accordingly.
model_level_number_at_base_of_ocean_mixed_layer_defined_by_sigma_theta
ocean_mixed_layer_thickness 
ocean_mixed_layer_thickness_defined_by_mixing_scheme
ocean_mixed_layer_thickness_defined_by_sigma_t 
ocean_mixed_layer_thickness_defined_by_sigma_theta 
ocean_mixed_layer_thickness_defined_by_temperature 
square_of_ocean_mixed_layer_thickness_defined_by_sigma_t
d. Martin Juckes provided a definition for transformed_eulerian_mean and this was added to two existing names: 'The "Transformed Eulerian Mean" refers to a formulation of the mean equations which incorporates some eddy terms into the definition of the mean, described in Andrews et al (1987): Middle Atmospheric Dynamics. Academic Press.'
northward_transformed_eulerian_mean_air_velocity
eastward_transformed_eulerian_mean_air_velocity
