Dear All,

The standard name table has been updated. The current version is now version 68, dated 18th September 2019. The changes have also been published on the NERC vocabulary server and are listed at the end of this message. A KWIC index for version 68 has been added to the CF website - these will now be generated routinely as part of the standard name updates. The changes are listed at the end of this message.

Cathy Smith has been added to the list of standard name contributors: http://cfconventions.org/Data/cf-standard-names/docs/standard-name-contributors.html.

Apologies for the delay in getting this version onto the website - this is due to my switching to a new work laptop and taking a few days to get organised again. The next update to standard names is planned for 14th October.

Best wishes,
Alison 

---
V68 standard name changes
1. New names
a. One new name of quality_flag (canonical units: 1) proposed by Ken Kehoe
b. One new name of sea_water_temperature_at_sea_floor (canonical units: K) proposed by Cathy Smith and Andrew Barna.
2. Aliases
No new aliases were added in this update.
3.  Modifications to definitions of existing names
a. The definition of status_flag was updated to cross-reference the new name quality_flag and clarify that status_flag can refer to instrument status as well as data quality information.
b. A typographical error was corrected in the definition of volume_fraction_of_frozen_water_in_soil.
