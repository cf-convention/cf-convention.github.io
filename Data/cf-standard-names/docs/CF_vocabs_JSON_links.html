Links to JSON format versions of CF controlled vocabularies

The following links will extract JSON format vocabulary lists from the NERC Vocabulary Server (NVS) using the SPARQL endpoint. The output is text that can be viewed in a browser window or text editor, or optionally downloaded and saved to your local file system. The links extract the current versions of the vocabulary lists. N.B. The link to the standard names currently also fetches aliases - these are flagged as "deprecated" terms in the NVS.
