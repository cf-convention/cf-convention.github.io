Hi Rich -

The website transition is progressing fairly well - I'm working closely
with Brian Eaton to make sure things go smoothly.
I've created a "Website Transition" milestone in Trac so folks can stay
up-to-date. Clicking the "closed tickets" or "active tickets" links will
allow you to drill-down:
http://www-pcmdi.llnl.gov/cf/trac/milestone/Website%20Transition

There is also now a "Website Transition" page in Plone which basically
just links to the Trac milestone.

A few more things need to happen before the CF discussions can be moved
to the Trac ticketing system, mainly setting up a series of mailing
lists and Trac components corresponding to the various working groups
identified at GO-ESSP.

The CF Conventions document and Standard Name Table have both been
ported to the new website, and Brian will be performing a detailed
comparison of the "old" and "new" versions of the document to make sure
it is accurate:
http://www-pcmdi.llnl.gov/cf/documents

The DocBook XML and "build" scripts are now in Subversion:
http://www-pcmdi.llnl.gov/cf/trac/browser

Some of the supporting documentation and advanced search features of the
standard name table have yet to be translated/implemented...

So to answer your question, we now have a roadmap, but there are not any
dates attached. My hope is we can have the new system functioning by the
end of August.

Thanks,
Kyle
