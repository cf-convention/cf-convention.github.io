> New aliases to be added: (proposed by Jonathan Gregory)
>
> For consistency with the new CEOP-II model name of
> volume_fraction_of_condensed_water_in_soil_pores all the existing
> "volume_fraction_of_water_in_soil..." names will be changed to use
> the wording: "volume_fraction_of_condensed_water_in_soil..." and
> aliases to the old names will be added. This will affect four
> names.
>
> Alias to be removed: (proposed by Roy Lowry)
>
> Currently there are two names,
> surface_downwelling_longwave_flux_in_air
> and
> surface_downwelling_longwave_flux_in_air_assuming_clear_sky,
> both of which are aliased to the older name of
> surface_downwelling_longwave_flux. This is deemed to be an error
> in the standard name table and may lead to confusion, for example,
> in Roy's work on ontology building. Therefore, the alias will be
> removed from
> surface_downwelling_longwave_flux_in_air_assuming_clear_sky.
