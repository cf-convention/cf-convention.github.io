Dear All,

I am intending to update the standard name table on Tuesday 17th July. The update has been delayed from 10th July to allow time for any last minute corrections or objections to the planned changes.

New names to be included:

Geostrophic current names (proposed by Olivier Lauret):
surface_geostrophic_sea_water_x_velocity (m s-1)
surface_geostrophic_sea_water_y_velocity (m s-1)
surface_geostrophic_sea_water_x_velocity_assuming_sea_level_for_geoid
(m s-1)
surface_geostrophic_sea_water_y_velocity_assuming_sea_level_for_geoid
(m s-1)

Climate model variable names (proposed by Heinke H?ck):
integral_of_surface_downward_latent_heat_flux_wrt_time; W s m-2
integral_of_surface_downward_sensible_heat_flux_wrt_time; W s m-2
integral_of_surface_net_downward_longwave_flux_wrt_time; W s m-2
integral_of_surface_net_downward_shortwave_flux_wrt_time; W s m-2
integral_of_toa_net_downward_shortwave_flux_wrt_time; W s m-2
integral_of_toa_outgoing_longwave_flux_wrt_time; W s m-2
surface_net_downward_radiative_flux; W m-2

Spelling correction to a recently added name:
The name
mole_fraction_of_hexachlorbiphenyl_in_air
was added at the last update on 15th May. It will be corrected to:
mole_fracton_of_hexachlorobiphenyl_in_air.

Best wishes,
Alison
