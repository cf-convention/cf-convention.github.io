Dear All,

The standard name table has been updated with the changes previously
announced. The current version of the table is version 8, dated 15
April 2008. There are 136 new names, 6 new aliases and modifications to
47 descriptions. Another update to the standard name table is due to
take place in May.

Best wishes,
Alison
