Dear All,

I plan to do an update of the standard name table next Tuesday, 10th
June. This will include a relatively small number of changes, namely
the SST names and some aliases for the recently introduced CCMVal names.
The next scheduled update for the standard name table is will be on
Tuesday 8th July.

1. New names to be included in the standard name table.

sea_surface_skin_temperature; K
sea_surface_subskin_temperature; K
sea_surface_foundation_temperature: K

2. Aliases to be introduced for existing standard names.

eastward_transformed_eulerian_mean_velocity ->
eastward_transformed_eulerian_mean_air_velocity

northward_transformed_eulerian_mean_velocity ->
northward_transformed_eulerian_mean_air_velocity

northward_eliassen_palm_flux -> northward_eliassen_palm_flux_in_air

upward_eliassen_palm_flux -> upward_eliassen_palm_flux_in_air

northward_heat_flux_due_to_eddy_advection ->
northward_heat_flux_in_air_due_to_eddy_advection

upward_flux_of_eastward_momentum_due_to_nonorographic_eastward_gravity_w
aves ->
upward_eastward_momentum_flux_in_air_due_to_nonorographic_eastward_gravi
ty_waves

upward_flux_of_eastward_momentum_due_to_nonorographic_westward_gravity_w
aves ->
upward_eastward_momentum_flux_in_air_due_to_nonorographic_westward_gravi
ty_waves

3. Spelling correction to a recently introduced standard name.

mole_fraction_of_methyl_chloride _in_air should be
mole_fraction_of_methyl_chloride_in_air, i.e., without the blank.
Thanks to Veronika for pointing this out.

Best wishes,
Alison
