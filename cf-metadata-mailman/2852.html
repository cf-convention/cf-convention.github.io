Dear All,

The standard name table has been updated to version 16, dated 11 October
2010.

I made a few last minute changes to the published list of names. In the
end I did not add
sea_ice_albedo; 1
tendency_of_surface_snow_melt_amount; kg m-2 s-1.
The first was already in the table (sorry for not spotting it earlier)
and the second is redundant because it is defined in the same way as the
existing name
surface_snow_melt_flux; kg m-2 s-1.

I made changes to two of the carbon cycle names (see separate posting
for more details) which I believe simplify the wording and also more
accurately reflect their meaning:
surface_net_downward_mass_flux_of_carbon_dioxide_expressed_as_carbon_due
_to_photosynthesis_and_respiration_and_fires_including_land_use_change;
kg m-2 s-1
became
surface_net_downward_mass_flux_of_carbon_dioxide_expressed_as_carbon_due
_to_all_land_processes; kg m-2 s-1;
surface_net_downward_mass_flux_of_carbon_dioxide_expressed_as_carbon_due
_to_photosynthesis_and_respiration_and_fires_excluding_land_use_change;
kg m-2 s-1
became
surface_net_downward_mass_flux_of_carbon_dioxide_expressed_as_carbon_due
_to_all_land_processes_excluding_anthropogenic_land_use_change; kg m-2
s-1.

Additionally, I made sure that all names referring to land use change
use the phrase anthropogenic_land_use_change consistently:
carbon_content_of_products_of_land_use_change; kg m-2
became
carbon_content_of_products_of_anthropogenic_land_use_change; kg m-2
surface_upward_mass_flux_of_carbon_dioxide_expressed_as_carbon_due_to_em
ission_from_fires_excluding_land_use_change; kg m-2
became
surface_upward_mass_flux_of_carbon_dioxide_expressed_as_carbon_due_to_em
ission_from_fires_excluding_anthropogenic_land_use_change; kg m-2.

All other names have been added as previously notified.

Best wishes,
Alison
