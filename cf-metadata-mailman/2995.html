Dear all,

  My next question will concern aerosols.

  I'm looking for a CF standard name for aerosol size distributions
expressed in numbers concentrations.

  "number_concentration_of_ambient_aerosol_in_air" is the name of the
physical parameter, but I also need to express the fact that I have
partial counts for a list of size classes.

  My first guess would have been
"size_distribution_of_number_concentration_of_ambient_aerosol_in_air",
but, in the "Guidelines for Construction of CF Standard Names", The most
appropriate transformation seems to be : "histogram_of_X[_over_Z]".

   So, I come to :
"histogram_of_number_concentration_of_ambient_aerosol_in_air_over_size".

But : http://puma.nerc.ac.uk/cgi-bin/cf-checker.pl reports :

Checking variable: mic_conctab_SPP200_sync_1
------------------
ERROR (3.3): Invalid standard_name: histogram_of_number_concentration_of_ambient_aerosol_in_air_over_size


   Did I get it wrong ? Or has anybody ever used another name for the
same quantity ?

Best Regards,

Bruno.
PS : same story for
histogram_of_number_concentration_of_cloud_liquid_water_particles_in_air_over_size
