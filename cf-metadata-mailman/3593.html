Hi all,

I've been following this thread with great interest. For me it boils down to this question:

 - Is the datum always known by the data provider?

If the answer is "yes" then I see no reason to omit the datum (and plenty of reasons to include it). If there *are* situations where the datum is genuinely unknown or undefined then we need to express this clearly too so users can beware.

(Previous posts mentioned GCMs as possible situations where the datum is unknown - is this really true? Surely there is always a sphere with a known radius on which coordinates are based?)
