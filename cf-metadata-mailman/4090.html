Dear All,

In August Jim Biard requested one new standard name of

brightness_temperature_anomaly

which is defined as the difference between a brightness_temperature and its climatological value. As noted by Philip Cameron-Smith and Jonathan Gregory, this name follows established syntax and the use of "anomaly" is consistent with other similar names. There have been no other comments on this name and it is accepted for inclusion in the standard name table. The canonical units of this quantity will be Kelvin (K). The name will be added at the next update of the standard name table.

Best wishes,
Alison
