Dear All,

In February Richard Hattersley pointed out that a number of standard names had trailing white space in the XML version of the standard name table:


atmosphere_moles_of_acetic_acid

atmosphere_moles_of_alpha_hexachlorocyclohexane

tendency_of_atmosphere_mass_content_of_nitrogen_monoxide_due_to_emission

tendency_of_potential_energy_content_of_ocean_layer_due_to_diffusion.



The extra white space was also present (although not visible!) in the HTML file which is generated directly from the XML. It seems to have been weeded out of the standard names list on the NERC vocabulary server. I have fixed the XML and HTML files on the CF website for the latest version of the standard name table.



Thanks to Richard for raising this.

Best wishes,
Alison
