Hi.


If you are going to use the scalar method, you must include a "coordinates"
attribute on each data variable that names the scalar variables that specify the
coordinates for the data variable. So, you should add an attribute named
coordinates with the value "time longitude latitude" to temperature,
temperature_whole_profile_flag, temperature_qc_flag, temperature_sd_test,
salinity, etc.
