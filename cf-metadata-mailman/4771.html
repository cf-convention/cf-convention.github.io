Hi all,

We're working at the moment on converting our cloud products (cloudmask,
cloudtype, ...) to netcdf with cf conventions.

Within this context, we would like to associate a palette of (rgb)
colors to the data, which are the usual colors we use to represent this
type of data (you can have a look here to get an idea:
http://www.nwcsaf.org )

However, I can't find information on how to do this in netcdf and more
particularly within CF.

Is there any relevant document already describing this issue?

Otherwise, I guess one would have could define a new dimension
corresponding to the number of discrete classes in the data and
associate with an rgb triplet... Or what would you recommend?
