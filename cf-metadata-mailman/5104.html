Dear Gisbert

You don't need to define a new standard name. This can be described by using
a coordinate variable of radiation_wavelength. If there is a single band of
radiation, the dimension will have size 1 for the coordinate variable, and
you would put 100 and 400 nm as the bounds. The coordinate value is nominal
and arbitrary; the mid-point (250 nm) would probably be a good choice.

Best wishes

Jonathan
