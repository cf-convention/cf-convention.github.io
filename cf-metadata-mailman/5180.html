Dear All,

The standard name table was updated earlier today with the list of changes detailed below. The current version of the table is now version 23, dated 23 March 2013. The changes have also been published on the NERC vocabulary server.

John Maurer and Andreas Hilboll have been added to the list of standard name contributors (http://cf-pcmdi.llnl.gov/documents/cf-standard-names/cf-standard-name-contributors). (This month's other contributors are already listed).

Please note that there are also a number of open mailing list threads in which standard names are currently under discussion. Those proposals are not included in today's update. The standard name table will be updated again in approximately one month's time and names that are formally accepted in the intervening period will be included in the next update.

Best wishes,
Alison

1. New names

a. sea_water_turbidity (1). Proposed by John Maurer.

b. Stratospheric/tropospheric trace gas column amounts. Proposed by Andreas Hilboll.

stratosphere_mole_content_of_nitrogen_dioxide (mol m-2)
troposphere_mole_content_of_nitrogen_dioxide (mol m-2)
troposphere_mole_content_of_glyoxal (mol m-2)
troposphere_mole_content_of_formaldehyde (mol m-2)
troposphere_mole_content_of_iodine_monoxide (mol m-2)
troposphere_mole_content_of_bromine_monoxide (mol m-2)
troposphere_mole_content_of_sulfur_dioxide (mol m-2)
troposphere_mole_content_of_ozone (mol m-2)

c. Ocean volume transport names. Proposed by Matthias Lankhorst.

ocean_volume_transport_across_line (m3 s-1)
ocean_volume_x_transport (m3 s-1)
ocean_volume_y_transport (m3 s-1)

2. Modifications to definitions of existing names

a. secchi_depth_of_sea_water
Added cross-reference to new sea_water_turbidity name

b. sea_water_pressure
Added cross-reference to sea_water_pressure_due_to_sea_water name added at Version 22 of standard name table

3. Correction of an error in the standard name table

In the update from version 17 to 18 of the standard name table the existing name tendency_of_atmosphere_water_vapor_content_due_to_advection should have been turned into an alias of the new version tendency_of_atmosphere_mass_content_of_water_vapor_due_to_advection. However, the name was accidentally deleted and the new version was never added. In the NERC vocabulary server, the old version of the name was still listed in versions 18-22. Both the CF and NERC listings of the standard names have been corrected from version 23 onwards.
