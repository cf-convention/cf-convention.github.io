Dear All,

I have published an updated version of the standard name table on the new CF github website. This is version 27 - please see the link from http://cfconventions.org/documents.html. Currently only the XML document is available but I hope to make the HTML version available shortly. Version 27 is dated 28 November 2013. This was the originally intended publication date but it did not go ahead as planned due to technical problems with the old web site. I am choosing to stick with the original date stamp for consistency with the NERC vocabulary server where the changes did get published in November.

There is now a large backlog of standard names and I apologise for the lack of progress in recent months, partly due to the web site changes and partly due to my working on another project for a time. I would like now to get back to the routine of updating the standard name table every 1 - 2 months and over the next several updates I will be working through the backlog until we are caught up again. I think it will be better to break the task into manageable chunks rather than delay publication until all the names are ready.

Best wishes,
Alison
