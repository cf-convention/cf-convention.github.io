We of the Marine Metadata Interoperability community (OK, mostly Carlos) have produced a prototype tool for browsing and searching CF standard names. This is a simple, user-oriented search service for the CF standard names. It uses the most recent CF Standard Names vocabulary as loaded into the MMI Ontology Registry and Repository, and is internally driven by the associated SPARQL endpoint.

Please try out the prototype at http://mmisw.org/experimental/cfsn. Feedback is welcome; either email Carlos Rueda (copied) and myself, or provide feedback on the github space for the tool [2] (the issues page [3] and wiki [4] are available there too).

The in-progress CF FAQ [5] will include a link to any known CF standard name tools [6], including this one. Please contact me to help make that list complete.
