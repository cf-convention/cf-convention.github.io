Dear All,

The standard name and area type tables have both been updated today. The list of changes is detailed below.

The current version of the standard name table is now 29, dated 8 July 2015, and for area types it is version 3, also dated 8 July 2015. The links from http://cfconventions.org/standard-names.html and http://cfconventions.org/documents.html have been updated accordingly. The changes will also be published overnight on the NERC vocabulary server.

Best wishes,
Alison

2. Modification to definitions of existing names

atmosphere_optical_thickness_due_to_cloud (Canonical units: 1)
One sentence will be added to the definition of this quantity, as follows:
' "Cloud" means the component of extinction owing to the presence of liquid or ice water particles'

thermodynamic_phase_of_cloud_water_particles_at_cloud_top (Canonical units: 1)
The list of acceptable standardised strings has been expanded:
"cloud_top" refers to the top of the highest cloud. "Water" means water in all phases. A variable with the standard name of thermodynamic_phase_of_cloud_water_particles_at_cloud_top contains integers which can be translated to strings using flag_values and flag_meanings attributes. Alternatively, the data variable may contain strings which indicate the thermodynamic phase. These strings are standardised. Values must be chosen from the following list: liquid; ice; mixed; clear_sky; super_cooled_liquid_water; unknown.

runoff_flux (Canonical units: kg m-2 s-1)
Ethan Davis reported an error that has lain undiscovered in the standard name table since 2009! The definition of runoff_flux had accidentally been overwritten by that of an entirely different name, sea_surface_height_above_geoid. This was caused by human error (mine!) back in the days when I had to do a lot of hand editing of the table. The correct definition will be restored in V29.

Photosynthetic names:

3. Modification to units of existing names

Change to units of two names and clarify interpretation of practical salinity in the definitions.
The definitions of both names are being amended to underline the fact that the practical salinity scale should not be interpreted as either parts per thousand, or salt concentration in sea water in g kg-1.
sea_water_practical_salinity (Canonical units change from 1e-3 to 1)
change_over_time_in_sea_water_practical_salinity (Canonical units change from 1e-3 to 1)

area_type (Canonical units change from 1 to no units)
region (Canonical units change from 1 to no units)

4. New aliases for existing standard names

No new aliases are included in this update.

5. New area types

The following new area types have been added:
snow_free_land; fire; smoke; dust_aerosol; volcanic_ash_cloud; rain.
