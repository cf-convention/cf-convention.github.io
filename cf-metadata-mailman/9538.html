Dear All,

The standard name table has been updated. The current version is now Version 63, dated 5 February 2019. The changes have also been published on the NERC Vocabulary Server. No new names or aliases have been added in this update, but as mentioned last time I have been making corrections to a list of typographical errors in standard name definitions that were reported to me by Martin Juckes. Approximately half the typos have now been fixed - another batch will be included in the February update. All changes are listed at the end of this message. The next update is planned for 4th March.

Best wishes,
Alison 

---
