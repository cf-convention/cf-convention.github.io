Dear All,

The standard name table has been updated. The current version is now version 65, dated 9th April 2019. The changes have also been published on the NERC Vocabulary Server. All changes are listed at the end of this message.

Dan Say has been added to the list of standard name contributors: http://cfconventions.org/Data/cf-standard-names/docs/standard-name-contributors.html. Katherine Pugsley and Ranjini Swaminathan have also been added for their contributions to version 64 of the table.

The next update to standard names is planned for 14th May.

Best wishes,
Alison 

---
V65 standard name changes
1. New names
Several sets of halocarbon and other trace gas names, proposed by Dan Say:
mole_fraction_of_chloroform_in_air (Canonical unit: 1)
mole_fraction_of_dichloromethane_in_air (Canonical unit: 1)
mole_fraction_of_hcfc124_in_air (Canonical unit: 1)
mole_fraction_of_hfc125_in_air (Canonical unit: 1)
mole_fraction_of_hfc134a_in_air (Canonical unit: 1)
mole_fraction_of_hfc143a_in_air (Canonical unit: 1)
mole_fraction_of_hfc152a_in_air (Canonical unit: 1)
mole_fraction_of_hfc227ea_in_air (Canonical unit: 1)
mole_fraction_of_hfc236fa_in_air (Canonical unit: 1)

2. New aliases
No new aliases were added in this update.
3. Modifications to definitions of existing names
Arising from the discussion of Dan Say's proposals, the definitions of the following names were updated to include the IUPAC name, tetrachloromethane:
atmosphere_mass_content_of_carbon_tetrachloride
atmosphere_moles_of_carbon_tetrachloride
mass_concentration_of_carbon_tetrachloride_in_air
mass_fraction_of_carbon_tetrachloride_in_air
mole_concentration_of_carbon_tetrachloride_in_air
mole_fraction_of_carbon_tetrachloride_in_air
tendency_of_atmosphere_mass_content_of_carbon_tetrachloride_due_to_emission
