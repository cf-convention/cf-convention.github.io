Compliance checker

This utility checks that a netCDF file which you supply complies with the CF conformance requirements and recommendations.

Conformance

Each version of the CF conventions document is accompanied by a conformance document which lists the requirements and recommendations that could be verified by the compliance checker or any program with a similar aim. For any recommendation which is not followed, the checker gives a warning; for any requirement which is not met, it reports an error.

Conventions preceding CF 1.0

The development of the CF convention built upon three earlier netCDF conventions: COARDS, NCAR CSM and GDT, as well as the NetCDF User Guide. Development of CF began in 1999, and there were several beta versions of CF before 1.0.
